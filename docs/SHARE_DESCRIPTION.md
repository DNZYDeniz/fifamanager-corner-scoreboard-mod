## Full description (download page)

### FM13 / FM26 Corner Stadium Scoreboard Mod

**FIFA Manager 13 · FIFA Manager 26** | Stadium Editor | Corner-type stadiums

#### Summary

Places **Video Screen (scoreboard)** elements on **corner** sections in the stadium editor for corner-type stadiums. Vanilla corners only allow small/large ground scoreboards, often in the wrong place (behind the roof, outside the stand).

#### Important — trade-off

This is **not** a complete scoreboard overhaul.

| Works | Breaks |
|-------|--------|
| Corner tribune sections | Behind-goal tribunes |
| | Main / marathon stands |
| | Other non-corner sections |

Only install if you need **corner** scoreboards. **Always back up** original files.

#### Features

- Unlocks all Video Screen sizes on corners
- Correct corner placement (in front of stands)
- Stable on corners (Large A placer; same visual size on corners)

#### Limitations

- Behind-goal and marathon scoreboards stop working correctly
- No free drag placement — editor snap rows only
- Replaces `StadiumDB.xml` + `Stadelems.big`

#### Compatibility

- **FIFA Manager 13** — supported  
- **FIFA Manager 26** — supported (same file path)

#### Install

1. Close the game  
2. Back up `data\stadium\generator\StadiumDB.xml` and `Stadelems.big`  
3. Copy mod files into that folder (overwrite)  
4. Delete old corner scoreboards and re-place in editor  

#### Uninstall

Restore your backup files.
